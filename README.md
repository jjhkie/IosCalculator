# IosCalculator

## [ 사용한 기술 ]

  - UIStackView
  
  - enum
  
    서로 관련된 값을 그룹으로 모아서 조금 더 쉽게 사용할 수 있도록 도와주는 타입
    
    enum 열거형이름 {
      
      case value1

      case value2

      case value3
   
   }
  
    열거형 이름은 대문자로 시작해야 하며, 한 줄에 하나씩 작성하는 방법도 있고, 한 케이스에 여러 값을 작성하는 방법도 있다.
